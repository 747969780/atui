## API


属性 | 说明 | 类型 | 默认值
-----|-----|-----|------
placeholder | 值为空时显示提示文字 | string | 请选择时间
disabledHours | 禁用的小时范围 | function | `function() { return [] }`
disabledMinutes | 禁用的分钟范围 | function | `function() { return [] }`
disabledSeconds | 禁用的秒钟范围 | function | `function() { return [] }`
