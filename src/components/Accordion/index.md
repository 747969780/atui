显示当前页面在系统层级结构中的位置，并能向上返回。

## 何时使用

- 当系统拥有超过两级以上的层级结构时；
- 当需要告知用户『你在哪里』时；
- 当需要向上导航的功能时。

## API


Accordion

| 参数      | 说明                              | 类型             | 默认值 |
|-----------|-----------------------------------|----------------|--------|
| openOne | 是否永远只展开一个面板 | Boolean | false


Panel

| 参数      | 说明                              | 类型              | 默认值 |
|-----------|-----------------------------------|-----------------|--------|
| @onToggle | 当展开或者关闭时触发的事件 | Function | function () {} |
