## API


属性 | 说明 | 类型 | 默认值
-----|-----|-----|------
hideOnClick | 是否在点击菜单项后隐藏菜单 ｜ `Boolean` | `true` |
offset | 下拉弹出层的偏移量, 如`[0, 50]`,横向不偏移,纵向偏移50px | `Array` | `[0, 0]` |
trigger | 显示下拉菜单的触发方式，可选值为`hover`, `click`, `focus`默认为`click` | `click` |
@change | 下拉菜单展示状态变化时触发 | `function(isOpen)` | 无 |
