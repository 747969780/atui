## 安装

```bash
$ tnpm install @ali/atui --save
```

## 示例

```jsx
<v-button>按钮</v-button>

new Vue({
  components: {
    vButton: atui.Button
  }
})
```

引入样式：

```jsx
// css引入 greater-blue | tao-orange | or tmall-red
import '@ali/atui/dist/greater-blue.css'



按需加载可通过此写法 `import { Alert } from '@ali/atui'`。
