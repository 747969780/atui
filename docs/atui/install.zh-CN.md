## 安装

```bash
$ tnpm install @ali/atui --save
```

## 示例

```jsx
<v-button>按钮</v-button>

new Vue({
  components: {
    vButton: atui.Button
  }
})
```

引入样式：

```jsx

<link rel="stylesheet" href="path/greater-blue.css">
```
> 注：目前支持三套皮肤，`greater-blue` ,`tao-orange`, `tmall-red`
