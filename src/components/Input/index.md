atui.Input

> 注意：input在HTML是原生标签，注册组件时请使用自定义组件比如vInput

## API



属性 | 说明 | 类型 | 默认值
-----|-----|-----|------
large | 按钮组尺寸，大 | Boolean | false |
small | 按钮组尺寸，小 | Boolean | false |
placeholder | 占位提示 | String | |
valid-status | 当前验证状态 有三个状态可选：`error`,`success`,`warn` | String | |
maxlength | 最大长度 | String | |
